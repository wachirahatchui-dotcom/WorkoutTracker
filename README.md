# WorkoutTracker

โปรแกรมสำหรับบันทึกข้อมูลการออกกำลังกาย พัฒนาด้วย C# Windows Forms

## เทคโนโลยีที่ใช้
- C# Windows Forms
- .NET
- Visual Studio

## วิธีดาวน์โหลดโปรเจกต์
1. เข้า GitHub Repository นี้
2. กดปุ่ม Code → Download ZIP
3. แตกไฟล์ ZIP ลงในเครื่อง

## วิธีเปิดและรันโปรแกรม
1. เปิด Visual Studio
2. เลือก Open Project/Solution
3. เปิดไฟล์ .sln
4. กด Start เพื่อรันโปรแกรม

## ส่วนประกอบหน้าจอ
- ExerciseID : รหัสกิจกรรม
- ExerciseName : ชื่อกิจกรรม
- Intensity : ระดับความหนัก (Low, Medium, High)
- Status : สถานะ (Planned, In Progress, Done)
- ปุ่ม Add : เพิ่มข้อมูลและแสดงสรุป
- ปุ่ม Clear : ล้างข้อมูล
- ปุ่ม Exit : ปิดโปรแกรม

## ตัวอย่างการใช้งาน
1. กรอก ExerciseID และ ExerciseName
2. เลือก Intensity และ Status
3. กดปุ่ม Add
4. โปรแกรมจะแสดง MessageBox สรุปข้อมูล

## การล้างข้อมูลและออกจากโปรแกรม
- ปุ่ม Clear ใช้ล้างข้อมูลทั้งหมดในช่องกรอก
- ปุ่ม Exit ใช้ปิดโปรแกรม
